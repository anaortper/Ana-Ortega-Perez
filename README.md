# Ana Ortega Pérez — Academic Portfolio

Static academic portfolio website ready for GitHub Pages.

## How to publish on GitHub Pages

1. Upload all files in this folder to the root of your GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select **main** and **/(root)**.
5. Save.

The site includes a public CV file without personal address or phone number.
