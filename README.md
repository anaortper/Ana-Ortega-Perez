# Ana Ortega Pérez — Academic Portfolio

Static academic portfolio website ready for GitHub Pages.

## Upload to GitHub Pages

1. Create a public repository.
2. Upload all files in this folder, not the folder itself.
3. Go to Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Save.

## Files

- `index.html` — website content
- `styles.css` — visual design
- `script.js` — small interactions and publication filters
- `Ana_Ortega_Perez_CV_Public.pdf` — downloadable public CV

## Editing

Most text can be edited directly in `index.html`.
